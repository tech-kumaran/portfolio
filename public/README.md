# Public Assets

This folder contains public assets for the portfolio.

## Resume

To add your resume:

1. **Create or prepare your resume as a PDF** - You can use:
   - Google Docs (File > Download > PDF)
   - Microsoft Word (File > Save As > PDF)
   - Canva
   - Any PDF editor

2. **Name the file** `resume.pdf` and place it in this folder

3. **The download button in the Contact section** will now work and download your resume

Example:
- File path: `/public/resume.pdf`
- Download name: `Kumaran-Resume.pdf`

## Quick PDF creation

If you need to create a resume PDF quickly:
- Use [Canva](https://canva.com) - Free resume templates
- Use [Google Docs](https://docs.google.com) - Free and easy
- Use [Overleaf](https://overleaf.com) - For technical resumes
